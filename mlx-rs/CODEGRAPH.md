## Summary

mlx-rs is a Rust binding/wrapper library for the MLX machine learning framework (Apple's array framework for ML). The project provides Rust APIs for tensor operations, neural network modules, and includes example applications demonstrating language model inference (Mistral, Qwen3) and MNIST digit classification training. It also contains procedural macro infrastructure for generating builder patterns and convenience macros for the library's API.
